Song Search

Search for a given song, artist, or genre using the given search bar. Songs will automatically be listed in alphabetical order.

Use the sort by features in the top left to change the sorting of the songs.

Find Songs Like

Search for songs closely related to a given song. This will generate the song's likeness score, which is based on criteria such as acousticness, tempo, and duration.

The top 20 most similar songs will automatically be listed in order of their likeness score. Use the sort by features in the top left to change the sorting of the songs.
